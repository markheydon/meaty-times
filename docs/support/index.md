# Support

If you need help or want to share feedback, there are a few simple ways to get involved.

## Raise an issue

If you find a bug or something does not look right, please open an issue on GitHub.

## Request an enhancement

If you have an idea for a new feature or a better experience, open an issue with a short description of what you would like to see.

## Contribute

Contributions are welcome. If you want to help improve the app, please review the repository contribution guidance and open a pull request with your changes.
