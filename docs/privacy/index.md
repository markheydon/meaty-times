# Privacy

Meaty Times is a free and open-source hobby project. The goal is to keep the experience simple and private.

## What this means in practice

- No user accounts are required.
- No personal data is stored by the app.
- No tracking cookies are used.
- No analytics are collected unless this is added in a future release.

## Future changes

If the project adds new features that collect data, the privacy information will be updated to reflect those changes.
