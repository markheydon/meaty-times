# Help

Meaty Times is designed to be easy to use, but a few pointers make it even simpler.

## Getting started

1. Open the roast calculator.
2. Choose the meat type you want to cook.
3. Enter the weight of the joint.
4. Choose the doneness you want where relevant.
5. Review the cooking time, target temperature, and serve-at guidance.

## Selecting meat types

The calculator covers common roast meats, including beef, lamb, pork, chicken, and gammon. Pick the type that matches your joint so the guidance is tailored to it.

## Cooking temperature guidance

Use the suggested internal temperatures as a guide and check the meat at the end of the cooking time. Poultry and pork can be served safely with the recommended doneness settings, while beef and lamb often vary by how well done you prefer them.

## Timing calculations

The timing guidance is designed to help you plan when to start cooking. If you already know when you want to serve the meal, the calculator can work backwards to suggest a start time.

## Frequently asked questions

### Do I need to create an account?

No. Meaty Times is a simple app for planning and cooking roast meals without signing up.

### Can I use it on my phone?

Yes. The interface is designed to work well on mobile and desktop screens.

### What if I am cooking a different size joint?

Enter the meat weight as accurately as you can. The calculation is based on that input to give a more helpful estimate.
